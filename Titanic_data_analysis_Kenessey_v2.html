Titanic data - investigate a dataset¶

Titanic was a British passenger liner that sank in the North Atlantic Ocean in 1912, after colliding with an iceberg during her voyage from Southampton to New York City. Out of the estimated 2,224 passengers and crew aboard, more than 1,500 died, making it one of the deadliest maritime disasters in modern history.

Just to get an impression about the overall statistics I've checked the Wikipedia post about RMS Titanic, where I have found plenty of information about the passengers and crew aboard. Below I listed a few points which are related to our dataset.

  • Titanic's passengers numbered approximately 1,317 people: 324 in First Class, 284 in Second Class, and 709 in Third Class
  • Titanic had around 885 crew members on board
  • 869 (66%) were male and 447 (34%) female
  • There were 107 children aboard, the largest number of which were in Third Class
  • Third Class fares from London, Southampton, or Queenstown cost 7,5s while the cheapest First Class fares cost 23
  • 923 passengers boarded Titanic in Southampton, 179 First Class, 247 Second Class and 494 Third Class
  • In Cherbourg 274 additional passengers were taken aboard, 142 First Class, 30 Second Class, and 102 Third Class
  • 123 passengers boarded Titanic in Queenstown, 3 First Class, 7 Second Class and 113 Third Class.

In our dataset we have demographics and passenger information about passangers and crew on board of the Titanic. It is impressive to see that they collected these statistics more than 100 years ago and we have information about 891 people from the estimated 2224 passangers and crew aboard. We have to work with this dataset, so let's have a closer look on our data, first of all here the descriptions of the header information:

Data Dictionary¶

Variable Definition Key and notes
survival Survival 0 = No, 1 = Yes
pclass Ticket class 1 = 1st, 2 = 2nd, 3 = 3rd, 1st = Upper 2nd = Middle 3rd = Lower

sibsp # of siblings / spouses aboard the Titanic Sibling = brother, sister, stepbrother, stepsisterSpouse = husband, wife (mistresses and fiancs were ignored)
parch # of parents / children aboard the Titanic Parent = mother, fatherChild = daughter, son, stepdaughter, stepson. Some children travelled only with a nanny, therefore parch=0 for them.
ticket Ticket number
fare Passenger fare
cabin Cabin number
embarked Port of Embarkation C = Cherbourg, Q = Queenstown, S = Southampton

Investigation of Question 1¶

Was the "Women and child first" code of conduct applied?This code means that the lives of women and children were to be saved first in a life-threatening situation, typically when abandoning the ship, when survival resources such as lifeboats were limited. For this I have to check two main demografic groups: women and children, which in my dataset means that I need to create new categories: male, female and child. So first define child as under the age of 14.

In [28]:
def male_female_child(passenger):
    age, sex = passenger
    if age < 14:
        return 'child'
    else:
        return sex
    
titanic_data['New_passenger'] = titanic_data[['Age', 'Sex']].apply(male_female_child, axis=1)

Limitations of the analysis¶

We have to highlight the limitations of our analysis. First of all as discussed it in the first session we don't have information about all the passangers and we don't have any information when and how this database was collected. Even if we assume that these data is representative for the whole passenger group we do have missing values in Age, Cabin and Embarked columns.

In case of the age we could have impute the average for the missing values or we could delete the rows with missing values, but simply we left these out from our calculation,. With use average we would have more data to work with, but on the other hand also assume that we have a representative input without outliers. With deleting the rows we lost information. So both solution has it's limitations. In case of the age I have defined an age range, maybe this could be also adjusted to the historical definition.

On these data we couldn't use any statistical test, just found some tendencies. We have seen correlations, but it does not imply causation. Like maybe the higher survival rate in first class was just because of the fact that those passengers were closer to the deck where all the all the life boats were.

It would be good to have more detailed information in the family members category and dive deeper into those ones.
